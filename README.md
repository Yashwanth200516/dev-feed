# DevFeed

DevFeed is a social media web application developed using Django. The platform enables users to share posts, interact through likes and comments, and manage their profiles in a clean and responsive interface.

## Features

- User Authentication (Login & Logout)
- Create, Read, Update, and Delete (CRUD) Posts
- User Profile System
- Like and Unlike Posts
- Comment on Posts
- Search Posts
- Authorization and Access Control
- Responsive UI using Bootstrap

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap

## Learning Outcomes

This project helped me gain hands-on experience with:
- Django Framework
- Django ORM
- Authentication & Authorization
- Database Relationships (One-to-Many, Many-to-Many)
- CRUD Operations
- Frontend-Backend Integration
- Responsive Web Design
